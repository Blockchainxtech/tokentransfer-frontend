# Token Transfer Frontend

## Installation:

1. **Clone Repositorygit clone https://github.com/Blockchainxtech/tokentransfer-frontend.git**

2. **Use Node.js version 18.18:**
   Ensure that you have Node.js version 18.18 installed for this project.

3. **Build the Application:**
   After cloning the repository, use the following command to build the application:

```bash
npm run build
```
